Limit string length in Input tag

May 22, 2014

1 ng-model instead of value

Let’s start from something simple. In &lt;input&gt; tag, there are two ways to display defaut value:

html &lt;input type="text" id="resetTesting" value="Testing" /&gt;

html &lt;input type="text" id="resetTesting2" placeholder="Testing" /&gt;

So if you want to modify the value:

javascript function resetInput() { document.getElementById("resetTesting").value = "I'm reset!"; document.getElementById("resetTesting2").value = "I'm reset, too!"; }

You can find the example code here: http://jsfiddle.net/J7D55/.

But, what if we want to do this in AngularJS?

Obviously, it dosen’t work if a ng-model is given:

html &lt;input name="widget.title" ng-model="widget.title" value="Original value"&gt;

2 ng-readonly

If you want to keep the value of &lt;input&gt; uneditable, you can use ng-readonly. Here’s an example:

html Check me to make text readonly: &lt;input type="checkbox" ng-model="checked"&gt;&lt;br/&gt; &lt;input type="text" ng-readonly="checked" value="I'm Angular"/&gt;

You can try this example here: http://plnkr.co/edit/xLPEa3314pNeYTvW2gFZ?p=preview.
