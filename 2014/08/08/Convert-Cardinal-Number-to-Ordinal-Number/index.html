Convert Cardinal Number to Ordinal Number

Aug 8, 2014

Found a really useful function, which converts cardinal number to ordinal number, for example:

1 -&gt; 1st
2 -&gt; 2nd
3 -&gt; 3rd
4 -&gt; 4th
`</pre>

I should say, it’s really cute, really beautiful, here we go:

<pre>`function getOrdinal(n) {
  var s=["th","st","nd","rd"],
    v=n%100;
  return n+(s[(v-20)%10]||s[v]||s[0]);
}
