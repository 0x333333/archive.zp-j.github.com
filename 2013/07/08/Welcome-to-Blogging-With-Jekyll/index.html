Welcome to Blogging With Jekyll

Jul 8, 2013

Welcome to Jekyll - a simple, blog aware, static site generator.

<small>Zhipeng JIANG</small>

Welcome to visit my first blog in Github!

With the help of GitHub, GitHub Pages, Hover and Jekyll, zp-j.github.com is now up and running!

This project has been my first use of Jekyll and the Liquid template language and so far everything has been very straightforward.

Jekyll Bootstrap made it super easy to choose a theme and checkout a copy of the code to my personal GitHub page.

A few commands later and I’m blogging again for the first time in years and looking forward to sharing my personal and professional experiences, thoughts and projects with the rest of the world.
